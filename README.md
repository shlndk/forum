Опис: Форум, створений з використанням фреймворку Laravel. Реалізовано повноцінну систему обговорень із підтримкою
ролей та взаємодії користувачів.

Функціонал:
    Реєстрація та авторизація користувачів,
    Створення та перегляд постів,
    Написання коментарів до постів,
    Пошук постів за заголовком,
    Додавання постів до обраного,
    Система ролей (user / admin) з керуванням доступами (Laravel-permission)

Використані технології: PHP/Laravel, MySQL, Boostrap, Laravel-permission
